How to Run the System Yourself:

  Ensure your virtual environment is activated, then run the commands from your main project folder:

  1. To enter a new Casino Campaign:


   venv/bin/python3 -m CRYPTO_EXECUTION_ENGINE.crypto_casino_entry


  2. To run the continuous Hedging Daemon:


   venv/bin/python3 -m CRYPTO_EXECUTION_ENGINE.crypto_dynamic_hedger --> use a while loop if posible as well
